# 基础信息录入脚本

## 开场白

```
我来帮你创建 TA 的数字人格。回答下面几个问题，每个都可以跳过。
信息越详细，生成的人格越准——尤其是人设定位和 MBTI，能大幅提升准确率。
```

---

## 问题序列

### Q1：称呼/代号

```
你怎么称呼 TA？（昵称、代号或随便起个名字都行）

例：李诞 / 那个脱口秀演员 / KOL001
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（中文转拼音，英文直接小写）

---

### Q2：身份定位

```
用一句话描述 TA 的身份——领域、角色、受众画像，想到什么写什么，跳过也行。

例：脱口秀演员 30多岁 知识型KOL 受众是城市青年
例：时尚博主 20多岁 微博百万粉 主要受众是年轻女性
例：创业者 40岁 科技圈 受众是投资人/从业者
例：音乐人 25岁 独立音乐 受众是小众乐迷
```

解析字段：
- **领域**：脱口秀 / 科技 / 时尚 / 音乐 / 美食 / 教育 / 其他
- **角色定位**：KOL / 创作者 / 创业者 / 学者 / 艺人 / 其他
- **年龄/年龄段**：20-25 / 25-35 / 35-45 / 45+
- **受众画像**：年龄段、性别倾向、兴趣标签

---

### Q3：MBTI 与性格框架（可选）

```
TA 的 MBTI 类型是？知道认知功能栈或者九型人格更好。
想到什么写什么，不知道可以跳过。

例：ENTP Ne主导 九型7w8
例：INFJ 只知道INFJ
```

解析字段：
- **MBTI 类型**：16 种四字母类型
- **主导功能**（dominant）：Ni / Ne / Si / Se / Ti / Te / Fi / Fe
- **九型人格**（enneagram）：1-9，含翼型如 7w8

---

### Q4：公众形象标签

```
用一句话描述 TA 的公众形象——人设特质、表达风格、价值观关键词，
想到什么写什么，跳过也行。

例：毒舌 理性派 反讽式幽默 嘴上损你但观点有深度
例：温柔细腻 共情强 会讲故事 但偶尔emo
例：高冷装 知识密集型 不废话 撞到专业领域会变得很健谈
```

解析字段：
- **表达风格标签**：从标签库匹配
- **价值观关键词**：从用户描述提取
- **主观印象**：自由描述，保留原文

#### 表达风格标签库

**说话方式**：话痨 / 沉默系 / 简洁派 / 冗长型 / 反讽式 / 直球派 / 绕弯子

**情绪表达**：理性冷感 / 情绪充沛 / 表面冷静内里emo / 戏剧化 / 低能量感

**专业姿态**：知识权威型 / 平等交流型 / 说教倾向 / 经验分享型

**互动风格**：爱怼人 / 温柔包容 / 会自嘲 / 偶尔毒舌 / 始终礼貌

**价值观倾向**：实用主义 / 理想主义 / 反叛感 / 保守派 / 自由派

---

### Q5：不聊的边界（可选）

```
有什么话题 TA 明显不愿意聊或者会回避的？

例：不聊私生活 被问到感情话题会绕开
例：不聊政治 被问到了会沉默或转移
例：没有明显边界 可以聊很多
```

## 确认汇总

收集完毕后展示：

```
信息汇总：

  👤  {称呼}
  🎯  {领域} {角色定位} {年龄段}（若未填则省略）
  👥  受众：{受众画像}（若未填则省略）

  🧠  MBTI：{mbti}（主导功能：{dominant}）（若未填则省略）
  🔢  九型：{enneagram}（若未填则省略）

  🏷️   表达风格：{expression_tags}（若未填则省略）
  💭  价值观：{values}（若未填则省略）
  🚫  边界话题：{boundaries}（若未填则省略）

确认无误？（确认 / 修改 [字段名]）
```

用户确认后进入 Step 2 访谈数据导入。

---

## Step 2 提示词（数据导入引导）

```
现在需要导入 TA 的访谈内容。

请粘贴访谈文字稿（可以直接粘贴文本）。

如果有多个访谈，可以分批追加。

跳过也行，后续随时可以追加（说"追加访谈"）。
```
